Google Drive Setup
Connect your Google account for seamless cross-device sync
How it works: ClearBooks saves your data as a JSON file inside a ClearBooks/ folder in your Google Drive. Sign in on any device with the same Google account and your full transaction history, budgets and settings load automatically.
1
Get a free Google Cloud Client ID
Go to console.cloud.google.com → Create project → Enable Google Drive API → Create OAuth 2.0 credentials → Add your site URL as authorised origin.
2
Paste your Client ID below
3
Sign in
Click the button below — you'll be asked to authorise read/write access to your Drive only.
📱 Using on Mobile

Save this HTML file to Google Drive, then open it in Chrome on Android or Safari on iPhone. Tap the share icon → "Add to Home Screen" — it works like a native app with full screen and offline access. Sign in with the same Google account for synced data.
